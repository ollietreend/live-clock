# Live Clock

A minimalist full-screen digital clock, built as a Progressive Web App.

**[ollietreend.github.io/live-clock](https://ollietreend.github.io/live-clock/)**

## Features

- 24-hour time display, updated every second
- Text scales dynamically to fill the screen
- Tap anywhere to enter fullscreen; exits on tap again
- Screen Wake Lock prevents the display sleeping while in fullscreen
- Installable as a PWA — works offline once installed

## Browser support

All modern browsers. Fullscreen and Wake Lock are not supported on older iOS Safari — the clock still works, those features are silently skipped.
